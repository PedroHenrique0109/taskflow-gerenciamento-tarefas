# TaskFlow

TaskFlow é um aplicativo simples para gerenciamento de tarefas desenvolvido em Flask e SQLite.

## Gestão de Mudanças

Durante o desenvolvimento foi identificada a necessidade de definir prioridades para melhorar a organização das atividades da equipe. Por isso, foi adicionado o recurso "Prioridade da tarefa" no cadastro e edição, garantindo que cada tarefa possa ser classificada como Baixa, Média ou Alta.
